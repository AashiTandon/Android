Violation of surveillance and privacy laws can result in:

### Criminal Penalties:
- **Prison time**: Up to 5 years federal, varies by state
- **Fines**: Up to $250,000 or more
- **Criminal record**: Felony convictions

### Civil Penalties:
- **Lawsuits**: Actual damages, statutory damages
- **Punitive damages**: Can be substantial
- **Legal fees**: Both sides
- **Restraining orders**: Protection orders against you

### Professional Consequences:
- **Employment termination**
- **Professional license revocation**
- **Reputation damage**
- **Difficulty finding future employment**

## DATA PROTECTION OBLIGATIONS

If you deploy this app, you MUST:

1. **Secure the data**
   - Encrypt all transmissions
   - Secure server access
   - Implement access controls

2. **Limit retention**
   - Delete data regularly
   - Don't keep data longer than necessary
   - Implement automatic deletion

3. **Protect from breaches**
   - Regular security audits
   - Monitor for unauthorized access
   - Incident response plan

4. **Respect data subject rights**
   - Allow data access requests
   - Enable data deletion
   - Provide data portability
