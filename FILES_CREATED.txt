NEXT STEPS:
===========

1. Start the server:
   cd server
   npm install
   npm start

2. Configure the Android app:
   Edit: app/src/main/java/com/example/datamonitor/network/DataUploader.kt
   Set: SERVER_URL = "http://YOUR_SERVER_IP:3000"

3. Build the Android app:
   ./gradlew assembleDebug
   OR open in Android Studio

4. Install and run on Android device

5. Grant all permissions when prompted

6. Verify data is being collected on server
